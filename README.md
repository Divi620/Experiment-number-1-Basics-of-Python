# **_1. INTRODUCTION TO PYTHON_**
Python is a high-level, interpreted, and general-purpose programming language. It was developed by Guido van Rossum and released in 1991. Python is known for its simple syntax and readability, which makes it easy to learn for beginners.
Python does not require complex syntax rules, making programs shorter and easier to understand compared to other programming languages. It is an interpreted language, meaning the code is executed line by line, which helps in easy debugging.
Python is widely used in engineering fields such as electronics, communication systems, automation, artificial intelligence, data science, embedded systems, and Internet of Things (IoT).

# 4. VARIABLES ðŸ§®

A variable is a name used to store data in the memory of a computer. Variables allow us to store values and use them later in a program. In Python, there is no need to declare the data type of a variable explicitly. The data type is automatically assigned based on the value stored in the variable.

While naming variables, certain rules must be followed. A variable name must start with a letter or an underscore. It should not begin with a number. Spaces are not allowed in variable names. Reserved keywords of Python cannot be used as variable names.

Data types define the type of data stored in a variable. Python supports several built-in data types. Integer data type is used to store whole numbers. Float data type is used to store decimal or fractional numbers. String data type is used to store text or characters. Boolean data type is used to store logical values such as True or False.

Understanding data types is very important because they determine the type of operations that can be performed on data in a program.
